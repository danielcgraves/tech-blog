# Tech Blog

## Description

### This project introduces the idea of the MVC model. We learned to work with sequelize and handlebars.js while keep code organized in multiple files using CRUD method in order to make all the files sync together.

## Installation

### Install requirements include: bcrypt, connect-session-sequelize, dotenv, express, express-handlebars, express-session, mysql2 and sequelize.
